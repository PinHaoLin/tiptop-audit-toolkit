# UI Layout Overflow Fix

## Why

步驟 7「人工確認」顯示稽核報告路徑時，路徑元件會被長字串與目前的寬度規則撐到主卡最寬，造成右側執行環境欄位視覺失衡，部分視窗尺寸也可能出現水平溢位。

## What Changes

- 限制稽核報告路徑元件只使用主卡可用寬度。
- 讓長路徑在元件內截斷或水平捲動，不得撐大外層 Grid 或 Card。
- 檢查同一套路徑元件在執行根目錄、aud、暫存路徑與本週資料夾等位置的行為一致。
- 保留現有流程功能與報告路徑資料，不變更後端產出邏輯。

## Scope

只涉及前端路徑顯示與響應式布局，不改變 Python 稽核流程、資料夾定位或 EXE 執行行為。
